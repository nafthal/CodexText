## Suggested usage notes
- Use one strong hero image + subtle gradient overlay for readability.
- Keep feature card images at the same aspect ratio (16:9 recommended).
- Compress locally (WebP/AVIF) before production use.
- Add `loading="lazy"` for below-the-fold images.
